Simple FizzBuzz programmers test

FizzBuzz is a simple programmers test which is designed to ensure that a programmer can create a simple
program, it works like this. The program will count to 100 and every number that is a divisible by
3, instead of printing the number it will print the word 'Fizz'. The program will do the same for
numbers divisible by 5 and print the word 'Buzz'. Numbers that are divisible by both will print FizzBuzz.
