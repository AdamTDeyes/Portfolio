Minecraft in Unity

Complex Terrian Generation

What is this project?

Ever wondered how minecraft managed to generate such a rich and interactive world without massive performance issues? Well I curentily did and so I put myself to work on learn how this is done, below is but a taste of some of my tests within the game engine Unity. Although its not the full minecraft title, this test shows the magic behind the famous terrian generation.

This project went into great detail about quads, textures and how rendering works when people aren't looking. The image above is a test which allow infite terrian generation. Just walk in a direction and the land would form in front of you.

The image above gives away a huge clue to how a game like minecraft can generate massive worlds without causing much lag. As shown above, everything is broken down into crunks (coillections of blocks).

My final test was to add and remove blocks. Although its only basic it shows the technique of how its done, and it's not done how we think. Interestingly each block add or removed becomes apart of the bigger game object known as a chunk. All this rerending and texture applying happens in fractions of time when we place a block.
